网站的意义

我一直喜欢记录生活，把自己的所见所闻所感用文字或图片的形式保存下来。我觉得这样可以让我更好地回顾和反思自己的成长，也可以让我更加珍惜每一天的点滴。我认为生活是一本书，每一页都值得被记录和分享。

除了记录生活，我还喜欢分享学习笔记。我是一个爱学习的人，我喜欢阅读各种书籍，学习各种知识，不断地充实自己。我觉得学习不仅是为了应付考试或者找工作，更是为了提高自己的素养和视野。我也喜欢和别人交流学习心得，听取不同的观点和建议，互相促进和进步。

因此，我决定建立这么一个个人网站，把我的生活记录和学习笔记都放在这里。我希望通过这个网站，可以和更多有相同兴趣和爱好的人交流和互动，也可以给其他想要记录生活或者分享学习的人一些启发和鼓励。我相信这个网站会成为我展示自己和认识世界的一个平台，也会成为我不断学习和进步的一个动力。

PWA

本网站适配了PWA，可以安装在设备上，然后以应用程序的形式出现在用户面前，成为用户设备的一个永久功能，用户可以像启动其他任何应用程序一样直接从操作系统启动它。

在桌面上：

  • Firefox 和 Safari 不支持在任何桌面操作系统上安装 PWA。
  • Chrome 和 Edge 支持在 Linux、Windows、macOS 和 Chromebook 上安装 PWA。

在移动设备上：

  • 在 Android、Firefox、Chrome、Edge、Opera 和 Samsung Internet Browser 上都支持安装 PWA。
  • 在 iOS 16.3 及更早版本上，PWA 只能通过 Safari 安装。
  • 在 iOS 16.4 及更高版本上，PWA 可以通过 Safari、Chrome、Edge、Firefox 和 Orion 的分享菜单安装

安装方式详见：安装和卸载 Web 应用 - 渐进式 Web 应用（PWA） | MDN
